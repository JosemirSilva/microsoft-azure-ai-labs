# prompts/ — Lab 5: IA Generativa

*Prompts* aplicados durante o lab, organizados por categoria.

## Convenção de nomes

```
{categoria}-{numero}-{descricao-curta}.md
```

## Categorias

| Prefixo | Categoria |
|---------|-----------|
| `texto-` | Geração de texto no Copilot ou Azure OpenAI |
| `imagem-` | Geração de imagem no Copilot |
| `codigo-` | Geração de código no Copilot ou Azure OpenAI |
| `chat-` | Conversa no Chat Playground do Azure OpenAI |
| `filtro-` | Teste de filtros de conteúdo |

## Estrutura de cada ficheiro de *prompt*

```markdown
# Nome do prompt

## Ferramenta
Microsoft Copilot / Azure OpenAI Chat Playground

## Prompt utilizado
(texto exato enviado)

## Resultado obtido
(resumo ou cópia da resposta — sem dados sensíveis)

## Observações
(comportamento, qualidade, limitações observadas)
```
