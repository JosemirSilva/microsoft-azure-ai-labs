# inputs

Coloque aqui as **imagens de entrada** utilizadas no laboratório.

## Direitos e uso

As fotos nesta pasta foram **obtidas na internet** e são utilizadas **apenas para fins de estudo** no âmbito deste laboratório (reconhecimento facial, OCR e análise de imagem no Vision Studio). Não há intenção de uso comercial; os direitos sobre as imagens originais permanecem com os respectivos titulares.

## Boas práticas

- Prefira imagens **sem dados pessoais sensíveis** ou use fotos com consentimento explícito / stock livre.
- Formatos comuns: **JPEG**, **PNG** (conforme aceite pelo Vision Studio).
- O enunciado da DIO fala em reconhecimento facial **e** resultados de texto nas imagens — pode usar imagens que contenham **rostos** e/ou **texto impresso** para exercitar ambos os cenários no **Vision Studio**.

Quando entregar o projeto, esta pasta deve conter as imagens que realmente processou.
